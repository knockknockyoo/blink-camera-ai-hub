Blink Camera AI Hub - Apple Silicon (M1/M2/M3/M4) Setup

1. Extract this ZIP on your personal Mac.
2. Double-click M1-1-Install.command. This is required once and needs internet access.
3. Double-click M1-2-Connect-Blink.command to connect your Blink account.
4. Optional: connect notifications with M1-3-Connect-Telegram.command.
5. Double-click M1-4-Start.command. The dashboard opens after about four seconds.

Use M1-5-View-Logs.command to follow analysis progress.
Press Control+C in the running terminal to stop the application.

Requirements
- macOS on Apple Silicon
- Python 3.10 through 3.13 (3.13 recommended)
- Node.js 22 or newer
- Internet access during initial setup and Blink synchronization

If macOS blocks a launcher, Control-click or right-click the file and select Open.

This distribution ZIP does not contain Blink credentials, Telegram tokens, or downloaded videos.
